This project is a portfolio app development using Next Js 16 (latest) app router version.

------------------------------------------------------------------------------

Objective - A portfolio site that is meant to showcase user professional profile i.e. projects, skills, contact along with demonstration of great, state of the art modern front-end, UI skills!

------------------------------------------------------------------------------

Since this project is utilising NextJs (TypeScript) framework, rules for routing, page , API creation has to follow standard NextJs App router conventions. In addition to that being a personal portfolio app, there would be requirenment for many static variables that would remain stay constant through out the project such as links for social handles i.e. github, linkedin, email, phone number etc. which would be defined/stored in contants.ts file at root directory.


Styling and UI components:

Important: Entire app should have light/dark mode enabled and should be fully responsive!

All pages, components has to be absolutely modern, beautifully designed, theme consistent, aligned with light/dark mode options. Transtion should be smooth, use react motion library for powerful animations

UI Components: This app would certainly require creation of many UI components, such as cards, list etc., create them all in a seprate components folder



Structure of the app:

App consist of following pages, with defined routes

- About page (/about)
- Projects page (/projects)
- Contact page (/contact)
- Skills page (/skills)
